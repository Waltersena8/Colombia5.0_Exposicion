Technical Reference

Glossary

More than 30 technical terms learned at Colombia 5.0. English · Español · Definition.

English Español Definition / Definición
Brief Resumen de proyecto Documento inicial que describe los objetivos, alcance y expectativas de un proyecto entregado por el cliente al equipo de desarrollo.
Prototype Prototipo Versión preliminar de un producto o mecánica que permite probar conceptos antes del desarrollo final.
Feedback Retroalimentación Respuesta o comentario sobre un producto o proceso que permite identificar áreas de mejora.
UX (User Experience) Experiencia de Usuario Conjunto de factores que determinan cómo un usuario interactúa y percibe un producto digital.
Delay Retraso Demora en la entrega o desarrollo de una funcionalidad respecto al cronograma inicial del proyecto.
Branching Ramificación narrativa Técnica en videojuegos y narrativas donde las decisiones del jugador generan caminos alternativos en la historia.
Occlusion Oclusión Técnica visual y narrativa que oculta elementos del entorno para generar misterio o controlar lo que el jugador percibe.
Visual Novel Novela Visual Género de videojuego que combina narrativa literaria con elementos visuales e interactivos, similar a un libro ilustrado interactivo.
Game Engine Motor de videojuego Software base que proporciona herramientas para desarrollar videojuegos (ej. Unity, Unreal Engine).
Scope Alcance del proyecto Límites y funcionalidades definidas que debe cubrir un proyecto de software o videojuego.
80/20 Rule (Pareto) Regla 80/20 (Pareto) Principio que indica que el 80% de los resultados provienen del 20% de los esfuerzos; aplicado para priorizar mecánicas clave.
Intellectual Property (IP) Propiedad Intelectual Creación original de un estudio (personaje, mecánica, historia) que le pertenece legalmente y puede ser comercializada.
Indie Studio Estudio independiente Empresa de desarrollo de videojuegos pequeña o mediana que opera sin el respaldo financiero de grandes publishers.
Ethical Hacker Hacker Ético Profesional de ciberseguridad autorizado para buscar vulnerabilidades en sistemas con el fin de corregirlas antes de que sean explotadas.
Cybersecurity Ciberseguridad Conjunto de prácticas, tecnologías y procesos diseñados para proteger sistemas, redes y datos de ataques digitales.
Vulnerability Vulnerabilidad Debilidad o falla en un sistema, aplicación o red que puede ser explotada para comprometer su seguridad.
Traffic Sniffing Interceptación de tráfico Técnica de ataque que captura paquetes de datos en una red para extraer información sensible transmitida sin cifrado.
Plain Text Texto claro Información transmitida o almacenada sin ningún tipo de cifrado, lo que la hace legible por cualquier interceptor.
Encryption Cifrado Proceso de transformar datos en un formato ilegible para protegerlos de accesos no autorizados durante su transmisión o almacenamiento.
Social Engineering Ingeniería Social Técnica de manipulación psicológica usada para engañar a personas y obtener información confidencial o acceso a sistemas.
Use Case Caso de Uso Descripción de cómo un usuario interactúa con un sistema para lograr un objetivo específico; usado en diseño de software y seguridad.
Abuse Case Caso de Abuso Escenario que describe cómo un atacante podría explotar un sistema; complemento de los casos de uso en análisis de seguridad.
Database Base de Datos Sistema organizado para almacenar, gestionar y recuperar información de forma estructurada.
Weak Password Contraseña débil Contraseña que puede ser adivinada o descifrada fácilmente por ser corta, predecible o reutilizada en múltiples servicios.
Public WiFi WiFi pública Red inalámbrica de acceso libre que representa un riesgo de seguridad al no garantizar cifrado entre usuarios.
Security by Design Seguridad desde el diseño Principio de desarrollo de software que integra medidas de seguridad desde las primeras etapas del diseño, no como un añadido posterior.
Phishing Suplantación de identidad Técnica de engaño digital que imita entidades confiables para robar credenciales o datos bancarios a los usuarios.
Recursion / Recursivity Recursividad En programación, proceso en que una función se llama a sí misma. En diseño creativo, reutilizar y recombinar ideas para generar nuevas soluciones.
Brainstorming Lluvia de ideas Técnica creativa grupal para generar gran cantidad de ideas sin juicios previos, con el objetivo de encontrar soluciones innovadoras.
Market Reference Referencia de mercado Análisis de productos similares en el mercado que sirven como punto de comparación para definir precio, calidad y posicionamiento.
Retention Retención de usuarios Capacidad de un producto o juego para mantener a los usuarios activos y comprometidos a lo largo del tiempo.
Roguelike Roguelike Género de videojuego caracterizado por niveles generados proceduralmente, dificultad alta y pérdida de progreso al morir.
Deck Builder Constructor de mazo Mecánica de juego donde los jugadores construyen y optimizan un conjunto de cartas o habilidades durante la partida.
NPC (Non-Player Character) Personaje no jugable Personaje controlado por la inteligencia artificial del juego, con quien el jugador puede interactuar pero no controlar directamente.
